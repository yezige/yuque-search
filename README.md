## 语雀快捷搜索插件

### 插件地址：

[Chrome Web Store](https://chrome.google.com/webstore/detail/yuque-helper/lffaohdkiepcnjnnbngjbmpgegbnepgf?hl=zh-CN)

### 使用方法：

- 右键插件 -> 选项 -> 设置`语雀 Token`
- 点击插件 -> 输入想搜索的关键字 -> 回车
- 再次回车，将在新标签页打开选中的文章
